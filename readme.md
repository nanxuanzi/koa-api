# 项目初始化

```shell
git init
touch .gitignore

npm init -y
```

# 构建工具

```sh
# webpack
yarn add -D webpack webpack-cli
touch webpack.config.js

# babel
yarn add -D @babel/core @babel/preset-env
yarn add -D babel-loader
touch .babelrc

# nodemon 
yarn add -D nodemon
touch nodemon.json

# 监听修改->打包->运行
```

# 搭建项目

```sh
# koa,koa-router
yarn add koa koa-router

# mongoose
yarn add mongoose
```
